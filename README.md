# Listen1 And
Listen1 And 是[listen1 chrome 插件](https://github.com/listen1/listen1_chrome_extension)基于chromium在安卓上的移植版。功能与插件版本完全一致，但是布局针对移动端优化。
# 特性
支持安卓手机与平板，平板在横屏时会布局会自动转为桌面布局（暂未实机测试）
